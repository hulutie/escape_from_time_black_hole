## English

**TimeHole** is a browser extension designed to help you break digital distractions and stay focused. Unlike traditional site blockers that simply show a "Blocked" page, TimeHole gently nudges you back to productivity by redirecting you to a site you've pre-approved.

-   **ğŸ§  Anti-Habitual Disabling**: Built-in "Challenge Mode" that requires you to type a randomly generated string to disable or pause protection, preventing impulsive disabling.
-   **ğŸ”„ Auto-Recovery**: If you manually disable protection, it can automatically re-enable itself after a set duration (e.g., 15 minutes).
